# TinyFusion Progress Log
Started: 2026-06-16

## Codebase Patterns
- TinyFusion is a Rust workspace with two crates: tinyfusion-core and tinyfusion-gui
- Core uses Axum for HTTP server, Tokio for async runtime
- GUI uses Tauri v2 with React frontend (to be added)
- Design system follows refined developer console aesthetic (dark-first, Inter font, Lucide icons)
